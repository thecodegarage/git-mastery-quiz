# Git Mastery Quiz 🎮

Interactive React quiz application with 200+ questions to test your Git knowledge!

## Features

- ✅ 200+ carefully crafted questions
- ✅ Filter by topic (Basics, Merge Conflicts, Rebasing, etc.)
- ✅ Multiple difficulty levels
- ✅ Instant feedback with explanations
- ✅ Progress tracking and scoring
- ✅ Dark mode support
- ✅ Responsive design

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Visit `http://localhost:5173` after running dev server.

## Topics Covered

- Git Basics & Daily Workflow
- Merge Conflicts
- Rebasing & Interactive Rebase
- History Rewriting (reset, revert, amend)
- Remote Workflows (push, pull, fetch)
- Recovery Operations (reflog, detached HEAD)
- Branching Strategies
- Stash & Cherry-Pick
- Debugging & Bisect

## Usage

### Select Topics
Choose one or multiple topics to focus your quiz session.

### Choose Difficulty
- 🟢 Beginner
- 🟡 Intermediate  
- 🔴 Advanced
- 🟣 Expert

### Take Quiz
Answer questions and receive immediate feedback with detailed explanations.

### Track Progress
See your score and review incorrect answers to improve.
